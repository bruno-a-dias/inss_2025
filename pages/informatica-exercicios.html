5.6 Exercícios de Informática

Simulado do Módulo!

Teste seus conhecimentos sobre todos os sub-tópicos de Noções de Informática. Resolva as questões como se estivesse na prova real!

Questões Integradas

Questão 1: Uma rede de computadores restrita a uma organização, como o INSS, que utiliza a tecnologia da Internet para compartilhar informações internamente, é corretamente denominada Extranet.

ERRADO

Justificativa: Uma rede interna e restrita a uma organização é uma Intranet. A Extranet seria uma extensão dessa rede para permitir o acesso controlado a usuários externos, como parceiros ou fornecedores.

Questão 2: No sistema operacional Windows, a combinação de teclas Shift + Delete exclui um arquivo permanentemente, sem enviá-lo para a Lixeira.

CERTO

Justificativa: A tecla "Delete" envia o arquivo para a Lixeira, permitindo a recuperação. A combinação "Shift + Delete" ignora a Lixeira e executa uma exclusão permanente, sendo uma ação que exige mais cautela do usuário.

Questão 3: Um Trojan (Cavalo de Troia) é um tipo de malware que se replica automaticamente através de redes, explorando vulnerabilidades de segurança, sem a necessidade de um arquivo hospedeiro.

ERRADO

Justificativa: A descrição se refere a um Worm. Um Trojan é um malware que se disfarça de software legítimo para enganar o usuário e ser executado, abrindo portas para outras ameaças.

Questão 4: No Microsoft Excel, a fórmula =SE(A1>=60; "Aprovado"; "Recuperação") exibirá a palavra "Aprovado" se o valor na célula A1 for exatamente 60.

CERTO

Justificativa: O operador lógico ">=" significa "maior ou igual a". Como 60 é igual a 60, a condição A1>=60 é verdadeira, e a função SE retornará o valor correspondente à condição verdadeira, que é "Aprovado".

Questão 5: Ao enviar um e-mail para múltiplos destinatários, o uso do campo "Cc" (Cópia Carbono) garante que cada destinatário não saiba para quem mais a mensagem foi enviada.

ERRADO

Justificativa: O campo que garante essa privacidade é o "Cco" (Cópia Carbono Oculta). Ao usar o "Cc", todos os destinatários (dos campos "Para" e "Cc") podem ver a lista completa de quem recebeu a cópia.

Questão 6: O cache do navegador armazena informações de login e preferências do usuário em sites, enquanto os cookies armazenam elementos de páginas para acelerar o carregamento.

ERRADO

Justificativa: A descrição está invertida. Os cookies armazenam informações como preferências e dados de login. O cache armazena elementos de páginas (imagens, scripts) para acelerar o carregamento em visitas futuras.

Questão 7: A principal função de um firewall é atuar como uma barreira de rede, controlando o tráfego de dados e bloqueando acessos não autorizados.

CERTO

Justificativa: Essa é a definição correta. Um firewall funciona como um "muro de fogo" digital entre uma rede privada (como a Intranet do INSS) e uma rede pública (a Internet), filtrando pacotes de dados com base em um conjunto de regras de segurança.

Questão 8: No LibreOffice Writer, a ferramenta Mala Direta permite combinar um documento modelo com uma fonte de dados para criar correspondências personalizadas em massa.

CERTO

Justificativa: A funcionalidade da Mala Direta é a mesma tanto no Microsoft Word quanto no LibreOffice Writer. Ela é projetada para automatizar a criação de documentos personalizados, como cartas, etiquetas ou envelopes, para múltiplos destinatários.

Questão 9: Um ataque de Ransomware tem como objetivo principal roubar senhas e dados bancários do usuário de forma silenciosa.

ERRADO

Justificativa: O objetivo principal do Ransomware é extorquir dinheiro da vítima. Ele faz isso criptografando os arquivos e tornando-os inacessíveis, exigindo um pagamento (resgate) para fornecer a chave de descriptografia. O roubo silencioso de dados é mais característico de um Spyware ou Keylogger.

Questão 10: No Microsoft Excel, a referência de célula $A1 é considerada mista, pois trava a coluna "A" mas permite que a linha varie quando a fórmula é copiada.

CERTO

Justificativa: Exatamente. A referência é mista porque parte dela é absoluta (a coluna, indicada pelo $) e parte é relativa (a linha). Ao arrastar a fórmula para baixo, a linha mudará (para A2, A3, etc.), mas ao arrastar para o lado, a coluna "A" permanecerá fixa.
