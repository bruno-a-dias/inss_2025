5. Noções de Informática

Ferramenta Essencial!

No serviço público moderno, o conhecimento em informática é indispensável. O concurso do INSS avalia noções práticas que serão utilizadas no dia a dia do servidor.

Conceitos de Internet e Intranet

Redes de Computadores
Conceito Descrição Acesso
Internet Rede mundial de computadores, pública e irrestrita, que interliga milhões de dispositivos globalmente. Público
Intranet Rede privada de uma organização que utiliza a mesma tecnologia da Internet (protocolos, navegadores), mas é restrita aos seus membros. Restrito (interno à organização)
Extranet Extensão da Intranet que permite o acesso externo controlado a parceiros, clientes ou fornecedores, geralmente via login e senha pela Internet. Controlado (externo autorizado)

Sistemas Operacionais

O edital geralmente foca no Windows (versões 7 e 10). É o software fundamental que gerencia todos os recursos do computador (hardware e software).

Principais Funções do Windows
  • Área de Trabalho: Interface principal com ícones e atalhos.
  • Barra de Tarefas: Contém o menu Iniciar, ícones de acesso rápido e programas abertos.
  • Explorador de Arquivos: Ferramenta para gerenciar arquivos e pastas.
  • Painel de Controle: Central de configurações do sistema.
Atalhos Essenciais
  • Ctrl + C: Copiar
  • Ctrl + V: Colar
  • Ctrl + X: Recortar
  • Alt + Tab: Alternar entre janelas
  • Win + E: Abrir Explorador de Arquivos

Noções de Segurança da Informação

Ameaças Virtuais (Malware)

Malware (Software Malicioso) é um termo genérico para qualquer programa projetado para danificar ou realizar ações indesejadas em um sistema.

AmeaçaDescrição
VírusPrograma que se anexa a outros arquivos para se propagar, precisando de um "hospedeiro".
WormSemelhante ao vírus, mas se propaga automaticamente pelas redes, sem precisar de um arquivo hospedeiro.
Trojan (Cavalo de Troia)Disfarça-se de programa legítimo, mas executa ações maliciosas quando ativado.
SpywareColeta informações do usuário sem o seu consentimento.
RansomwareCriptografa os dados do sistema e exige um resgate (ransom) para liberá-los.

Ferramentas de Proteção

Antivírus
Detecta e remove malwares conhecidos.
Firewall
Filtra o tráfego de rede, bloqueando acessos não autorizados.
Backup
Cópia de segurança dos dados para recuperação em caso de perda.

Edição de Textos, Planilhas e Apresentações

O edital pode cobrar tanto Microsoft Office (Word, Excel, PowerPoint) quanto LibreOffice (Writer, Calc, Impress)[6, 8]. As funções são semelhantes.

A) Editor de Texto (Word / Writer)

Funções Principais: Formatação de texto (fonte, parágrafo), inserção de tabelas e imagens, verificação ortográfica, controle de alterações, mala direta.

B) Planilha Eletrônica (Excel / Calc)

Funções Principais: Cálculos com fórmulas e funções, criação de gráficos, classificação e filtragem de dados[1].

Fórmulas Essenciais:
  • =SOMA(A1:A5): Soma os valores no intervalo.
  • =MÉDIA(B1:B10): Calcula a média dos valores.
  • =SE(A1>10; "Aprovado"; "Reprovado"): Função condicional.
  • =PROCV(valor; matriz; col_índice; [procurar_intervalo]): Procura um valor em uma coluna.

C) Apresentações (PowerPoint / Impress)

Funções Principais: Criação de slides, inserção de texto e objetos, animações de transição, modo de apresentação, slide mestre[1].

Navegadores e Correio Eletrônico

A) Navegadores (Browsers)

Google Chrome, Mozilla Firefox, Microsoft Edge

Funções: Navegação por abas, favoritos, histórico, modo de navegação anônima/privada, cookies (pequenos arquivos que armazenam informações de navegação).

B) Correio Eletrônico (E-mail)

Conceitos
Campos de Endereçamento:
  • Para (To): Destinatário(s) principal(is).
  • Cc (Cópia Carbono): Envia uma cópia para outros destinatários, visível a todos.
  • Cco (Cópia Carbono Oculta): Envia uma cópia oculta; os outros destinatários não veem quem está em Cco.

Exercícios Comentados

Questão 1: Uma Intranet é uma rede de acesso público que utiliza os mesmos protocolos da Internet para conectar empresas diferentes.

ERRADO

Justificativa: Uma Intranet é uma rede privada e de acesso restrito a uma única organização[1]. A rede que conecta empresas diferentes de forma controlada seria uma Extranet.

Questão 2: Um worm precisa de um arquivo hospedeiro para se propagar, enquanto um vírus se replica automaticamente pelas redes.

ERRADO

Justificativa: A descrição está invertida. O vírus precisa de um hospedeiro (um arquivo executável, por exemplo) para se propagar. O worm é autônomo e se replica através de redes[12].

Questão 3: Para somar os valores das células de A1 até A10 em uma planilha eletrônica, a fórmula correta é =SOMA(A1:A10).

CERTO

Justificativa: A função SOMA utiliza o caractere de dois-pontos (:) para definir um intervalo contínuo de células[1]. A sintaxe apresentada está correta para somar todas as células de A1 a A10.

Questão 4: O campo Cco (Cópia Carbono Oculta) em um e-mail permite que todos os destinatários vejam para quem a cópia oculta foi enviada.

ERRADO

Justificativa: A principal característica do campo Cco é justamente a privacidade. Os destinatários nos campos "Para" e "Cc" não conseguem ver quem foi incluído no campo "Cco".

Questão 5: Um firewall de rede tem como principal função remover vírus e outros malwares de um computador.

ERRADO

Justificativa: A função de remover malwares é do antivírus. Um firewall atua como uma barreira de proteção, controlando o tráfego de dados entre a rede interna e a externa (Internet) e bloqueando acessos não autorizados[1].

Resumo dos Pontos Principais

Para memorizar:
REDES E SEGURANÇA:
  • Internet (Pública), Intranet (Privada), Extranet (Acesso Externo)[1].
  • Vírus (hospedeiro), Worm (autônomo), Trojan (disfarce), Ransomware (resgate).
  • Firewall (barreira), Antivírus (removedor), Backup (cópia).
APLICATIVOS:
  • Windows: Gerencia arquivos e configurações.
  • Excel/Calc: Fórmulas (=SOMA, =MÉDIA, =SE) e gráficos.
  • E-mail: Diferença entre Cc (visível) e Cco (oculto).
