6.2 Conectivos e Tabela-Verdade

O Coração da Lógica!

Os conectivos lógicos são as ferramentas que unem proposições simples para formar proposições compostas. Dominar a tabela-verdade de cada um deles é essencial para resolver a maioria das questões[5].

Os 5 Conectivos Lógicos

São operadores que conectam proposições simples. Cada um possui uma regra específica para definir o valor lógico da proposição composta resultante[9].

1. Conjunção (E)

Símbolo: ∧

Lê-se: "p e q"

Regra: Só é verdadeira (V) quando ambas as proposições são verdadeiras.

2. Disjunção Inclusiva (OU)

Símbolo: ∨

Lê-se: "p ou q"

Regra: Só é falsa (F) quando ambas as proposições são falsas.

3. Disjunção Exclusiva (OU... OU)

Símbolo: ⊻ ou ∨

Lê-se: "ou p ou q"

Regra: Só é verdadeira (V) quando os valores lógicos são diferentes.

4. Condicional (SE... ENTÃO)

Símbolo: →

Lê-se: "se p, então q"

Regra: Só é falsa (F) no caso "Vera Fischer" (V → F). Nos demais casos, é verdadeira.

(p é antecedente, q é consequente)

5. Bicondicional (SE E SOMENTE SE)

Símbolo: ↔

Lê-se: "p se e somente se q"

Regra: Só é verdadeira (V) quando os valores lógicos são iguais.

Tabela-Verdade Completa

A tabela-verdade mostra todos os possíveis valores lógicos de uma proposição composta, com base nos valores das proposições simples que a formam. O número de linhas é calculado por 2n, onde 'n' é o número de proposições simples.

p q Negação
(~p)		 Conjunção
(p ∧ q)		 Disjunção
(p ∨ q)		 Condicional
(p → q)		 Bicondicional
(p ↔ q)

V F F F V F F
F V V F V V F
F F V F F V V

Exercícios Comentados

Questão 1 (CESPE/CEBRASPE): Considere a proposição "Se o servidor foi diligente, então o processo foi arquivado". Se a proposição "o processo foi arquivado" for falsa, então a proposição composta é necessariamente falsa.

ERRADO

Justificativa: A proposição é uma condicional (p → q). Sabemos que q é Falsa. O valor lógico da condicional dependerá do valor de p. Se p for V (V → F), a condicional será Falsa. Mas se p for F (F → F), a condicional será Verdadeira. Como não podemos garantir o resultado, a afirmação de que é "necessariamente falsa" está incorreta.

Questão 2 (CESPE/CEBRASPE): A proposição "O servidor atendeu ao público ou não elaborou o relatório" é verdadeira se o servidor não tiver atendido ao público e tiver elaborado o relatório.

ERRADO

Justificativa: Vamos analisar os valores. "O servidor não atendeu ao público" → p é Falso. "O servidor elaborou o relatório" → ~q é Falso (pois q seria "não elaborou"). A proposição composta é uma disjunção (p ∨ q). Com p=F e q=F, a disjunção (F ∨ F) resulta em Falso. Portanto, a afirmação está incorreta.

Questão 3 (CESPE/CEBRASPE): Para que a proposição "Maria é médica e Pedro é dentista" seja falsa, é necessário que ambas as proposições simples que a compõem sejam falsas.

ERRADO

Justificativa: A proposição é uma conjunção (p ∧ q). A regra da conjunção diz que ela só é Verdadeira se AMBAS forem V. Portanto, para que a conjunção seja Falsa, basta que pelo menos uma das proposições seja falsa. Não é necessário que ambas sejam.
