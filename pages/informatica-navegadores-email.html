5.5 Navegadores e Correio Eletrônico

As Portas de Entrada da Internet

Os navegadores são os programas que nos permitem acessar a World Wide Web, enquanto o correio eletrônico (e-mail) continua sendo uma das principais ferramentas de comunicação formal no ambiente de trabalho.

A) Navegadores Web (Browsers)

Principais Navegadores e Conceitos

Os navegadores mais comuns em concursos são Google Chrome, Mozilla Firefox e Microsoft Edge.

Conceito Descrição Atalho Comum
Navegação por Abas Permite abrir várias páginas web em uma única janela do navegador. Ctrl + T (Nova Aba)
Favoritos / Marcadores Salva o endereço de uma página para acesso rápido no futuro. Ctrl + D
Histórico Registra todas as páginas visitadas em um determinado período. Ctrl + H
Cookies Pequenos arquivos de texto que os sites armazenam no seu computador para guardar informações sobre suas preferências, login, itens de carrinho, etc. -
Cache Armazena temporariamente elementos de páginas web (imagens, scripts) para acelerar o carregamento em visitas futuras. -
Modo Anônimo / Privado Inicia uma sessão de navegação que não salva histórico, cookies ou informações de formulários após o fechamento da janela. Ctrl + Shift + N (Chrome)
Ctrl + Shift + P (Firefox)
Atenção!

O Modo Anônimo/Privado NÃO torna você invisível na Internet. Seu provedor de internet e os sites que você visita ainda podem registrar sua atividade.

B) Correio Eletrônico (E-mail)

Conceitos e Estrutura de uma Mensagem

O e-mail é um serviço que permite compor, enviar e receber mensagens através de sistemas eletrônicos. No ambiente corporativo, é uma ferramenta de comunicação formal.

Campos de Endereçamento

Campo Nome Completo Função
Para (To) - Destinatário(s) principal(is) da mensagem.
Cc Cópia Carbono Envia uma cópia da mensagem para outros destinatários. Todos os envolvidos (Para, Cc) conseguem ver esta lista.
Cco Cópia Carbono Oculta Envia uma cópia oculta. Os destinatários em "Para" e "Cc" não sabem quem está em "Cco". Um destinatário em "Cco" também não vê os outros que estão em "Cco".

Protocolos de E-mail

SMTP
Simple Mail Transfer Protocol. Protocolo padrão para enviar e-mails.
POP3
Post Office Protocol 3. Protocolo para receber e-mails. Geralmente baixa as mensagens para o computador e as apaga do servidor.
IMAP
Internet Message Access Protocol. Protocolo para receber e-mails. Sincroniza as mensagens com o servidor, permitindo o acesso de múltiplos dispositivos.

Exercícios Comentados

Questão 1: Ao navegar no modo anônimo de um navegador, o histórico de navegação e os cookies não são salvos no computador local após o fechamento da janela.

CERTO

Justificativa: Essa é a principal funcionalidade do modo anônimo ou privado. Ele cria uma sessão temporária que é descartada ao final, não deixando rastros no dispositivo local (histórico, cookies, dados de formulários).

Questão 2: Ao enviar um e-mail, um usuário colocou o endereço de seu chefe no campo "Cco". O destinatário principal, no campo "Para", saberá que o chefe também recebeu a mensagem.

ERRADO

Justificativa: A função do campo "Cco" (Cópia Carbono Oculta) é justamente garantir que os demais destinatários ("Para" e "Cc") não saibam para quem a cópia oculta foi enviada.

Questão 3: O protocolo SMTP é utilizado tanto para enviar quanto para receber mensagens de correio eletrônico.

ERRADO

Justificativa: O SMTP (Simple Mail Transfer Protocol) é o protocolo padrão para o envio de e-mails. Para o recebimento, os protocolos mais comuns são o POP3 e o IMAP.
