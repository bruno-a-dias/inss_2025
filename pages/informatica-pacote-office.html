5.4 Pacote Office e Editores de Texto

Ferramentas do Dia a Dia

Os editores de texto, planilhas e apresentações são as ferramentas mais utilizadas no cotidiano do serviço público. Os editais podem cobrar tanto o Microsoft Office (Word, Excel, PowerPoint) quanto o LibreOffice (Writer, Calc, Impress). As funcionalidades são muito semelhantes.

A) Editor de Texto (Word / Writer)

Funções Principais

Ferramenta para criação e edição de documentos textuais, como ofícios, relatórios e memorandos.

Formatação:
  • Fonte: Alterar tipo, tamanho, cor, negrito (Ctrl+N), itálico (Ctrl+I), sublinhado (Ctrl+S).
  • Parágrafo: Alinhamento (esquerda, centro, direita, justificado), espaçamento, recuo.
  • Estilos: Conjuntos pré-definidos de formatação (Ex: Título 1, Título 2).
Inserção e Ferramentas:
  • Tabelas: Organização de dados em linhas e colunas.
  • Imagens e Formas: Elementos visuais para ilustrar o documento.
  • Mala Direta: Criação de documentos personalizados em massa (cartas, etiquetas) a partir de uma fonte de dados.
  • Verificação Ortográfica e Gramatical (F7): Ferramenta para corrigir erros.

B) Planilha Eletrônica (Excel / Calc)

Organização e Análise de Dados

Ferramenta para cálculos, organização de dados, criação de gráficos e análise de informações.

Fórmulas e Funções Essenciais

FunçãoDescriçãoExemplo de Sintaxe
SOMASoma os valores em um intervalo de células.=SOMA(A1:A10)
MÉDIACalcula a média aritmética dos valores em um intervalo.=MÉDIA(B1:B5)
SERetorna um valor se uma condição for verdadeira e outro se for falsa.=SE(A1>10; "Aprovado"; "Reprovado")
PROCVProcura um valor na primeira coluna de uma tabela e retorna um valor na mesma linha de uma coluna especificada.=PROCV(valor_procurado; matriz_tabela; núm_índice_coluna; [procurar_intervalo])
CONT.SEConta o número de células em um intervalo que atendem a um critério.=CONT.SE(C1:C20; ">50")

Referências de Célula

Referência Relativa (Ex: A1)

Muda automaticamente quando a fórmula é copiada para outra célula. É o padrão.

Referência Absoluta (Ex: $A$1)

Permanece fixa quando a fórmula é copiada. O símbolo "$" trava a coluna e/ou a linha. Atalho: F4.

C) Apresentações (PowerPoint / Impress)

Comunicação Visual

Ferramenta para criar apresentações visuais de slides para reuniões, treinamentos e palestras.

Elementos Principais:
  • Slides: "Páginas" da apresentação.
  • Layout: Organização dos elementos em um slide (título, texto, imagens).
  • Design / Tema: Conjunto de cores, fontes e efeitos que dão uma aparência consistente à apresentação.
  • Transições: Efeitos visuais que ocorrem ao passar de um slide para outro.
  • Animações: Efeitos aplicados a objetos individuais dentro de um slide.
  • Slide Mestre: Slide "modelo" onde alterações de design são aplicadas a todos os outros slides.

Exercícios Comentados

Questão 1: No Microsoft Excel, para fixar a referência de uma célula (torná-la absoluta) ao arrastar uma fórmula, deve-se usar o símbolo de cifrão ($), como em $B$2.

CERTO

Justificativa: Exatamente. A referência relativa (ex: B2) se ajusta conforme a fórmula é copiada, enquanto a referência absoluta ($B$2) permanece fixa. O cifrão é usado para travar a coluna ($B2), a linha (B$2) ou ambas ($B$2).

Questão 2: A ferramenta Mala Direta do Microsoft Word serve para corrigir automaticamente os erros de ortografia de um documento.

ERRADO

Justificativa: A Mala Direta é utilizada para criar documentos personalizados em massa (como cartas ou etiquetas) a partir de uma fonte de dados (uma planilha, por exemplo). A ferramenta para correção ortográfica é a "Verificação Ortográfica e Gramatical" (F7).

Questão 3: No PowerPoint, o Slide Mestre é utilizado para aplicar um design e formatação consistentes a todos os slides de uma apresentação de uma só vez.

CERTO

Justificativa: Essa é a principal função do Slide Mestre. Alterações feitas nele (como a inserção de um logotipo ou a mudança da fonte do título) são refletidas em todos os slides que utilizam aquele layout, garantindo uniformidade visual.
